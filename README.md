# BetFree

BetFree is a SwiftUI application designed to help users manage and overcome gambling addiction through a supportive and mindful approach.

## Features

- Daily check-ins and streak tracking
- Mindfulness tools and resources
- Progress tracking and analytics
- Beautiful, modern UI with a focus on user experience

## Requirements

- iOS 16.0+ / macOS 13.0+
- Swift 5.9+
- Xcode 15.0+

## Installation

This project uses Swift Package Manager for dependency management.

1. Clone the repository
2. Open the project in Xcode
3. Build and run

## Architecture

The project is split into multiple modules:

- BetFree: Main application module
- BetFreeUI: UI components and design system
- BetFreeModels: Data models and business logic
